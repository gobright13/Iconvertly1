B. Content Elements
Add these inside layout blocks:

Standard Elements:
Text: Styles include font, size, color, alignment, spacing

Title/Header: Prominent header text with style options

Image: Supports JPEG, PNG, GIF, WEBP; alt-text; sizing; corner radius; clickable with optional tagging on click

Button: Customizable CTA buttons with styling and tag-on-click features

Divider: Horizontal line with styling options

Spacer: Blank space separator

Social: Social icon links; customizable icons and styles

HTML: Insert custom HTML/CSS code blocks inline

Media: Embed video thumbnail link (YouTube/Vimeo)

Link Bar: Add a horizontal list of links (navigation bar style)

Text Wrap Image: Wrap text around an image element
